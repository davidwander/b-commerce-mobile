# Regras do Projeto para a IA do Cursor

Este arquivo contém as regras específicas do projeto que a IA do Cursor deve seguir ao gerar respostas e fazer alterações no código.

## Regras Gerais

- Sempre priorize a clareza e a legibilidade do código.
- Mantenha a consistência com o estilo de codificação existente no projeto.
- Utilize as convenções de nomenclatura definidas no projeto.
- Ao propor mudanças, forneça uma justificativa clara e concisa.

## Regras Específicas (Adicione suas regras aqui)

- [Exemplo] Ao adicionar novos componentes React Native, sempre inclua estilos `NativeWind`.
- [Exemplo] Todas as novas funcionalidades de autenticação devem usar o serviço `firebase.ts` em `utils/firebase.ts`.
